## Tech Stack

- **React + TypeScript**
- **TanStack Router** – file-based routing
- **React Query** – local data cache (IndexedDB)
- **Dexie** – IndexedDB wrapper
- **shadcn/ui** (Tailwind) – UI kit

## Run Locally

```bash
bun install     # or npm / pnpm / yarn
bun dev         # Vite dev server

```
